Console usage:
```lua
local console = loadstring(game:HttpGet("https://raw.githubusercontent.com/slaimmials/SimpleGui/refs/heads/main/console.lua"))()
console:Print("printed text")
console:Warn("warn text")
console:Error("error text")
```
